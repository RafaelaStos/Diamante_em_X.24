# Academia do Programador

## Diamante em X

Um programa que projeta um diamante com a letra "X", com entrada de números impares.

Funcionalidades

   - Validação de Entrada: O Diamante garante que apenas opções válidas sejam aceitas.


**Requisitos**

    .NET SDK (recomendado .NET 5.0 ou superior) para compilação e execução do projeto.
